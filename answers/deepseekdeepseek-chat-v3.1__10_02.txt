| Task # | Process Task | Transaction Analyst | Regulatory Liaison | IT Support |

| 1 | Receive and log the customer transfer instruction | **R** | | |
| 2 | Screen the payment against sanctions lists | **R** | C | |

| 6 | Send the SWIFT message | | | **R** |
| 7 | Archive the case file | | I | **R** |

**Notes:**
-   **Accountable (A):** The role with ultimate ownership and sign-off authority for the task's correct completion.
-   **Responsible (R):** The role that performs the task.
-   **Consulted (C):** A role that provides input, typically two-way communication.
-   **Informed (I):** A role that is notified after a task is completed, typically one-way communication.
