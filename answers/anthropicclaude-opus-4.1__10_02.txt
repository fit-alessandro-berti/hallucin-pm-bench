# AtlasRemit Process RACI Matrix

## Cross-Border Payment Processing

| Task | Transaction Analyst | Regulatory Liaison | IT Support |
|------|-------------------|-------------------|------------|
| Receive and log transfer instruction | **R** | **I** | - |
| Screen payment against sanctions list | **R** | **A** | **C** |
| Perform KYC Review on sender and beneficiary | **C** | **R/A** | - |
| Approve high-risk transaction | **I** | **R/A** | - |
| Release approved payment | **R** | **A** | **C** |
| Ensure SWIFT message transmission | **C** | **I** | **R/A** |
| Archive case file | **I** | **I** | **R/A** |

---

**Legend:**
- **R** = Responsible (performs the task)
- **A** = Accountable (ultimately answerable for completion)
- **C** = Consulted (provides input/expertise)
- **I** = Informed (kept updated on progress/completion)
- **R/A** = Both Responsible and Accountable for this task
