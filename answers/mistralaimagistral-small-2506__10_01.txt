Here is the RACI matrix for FinCo’s order-to-cash workflow using the correct terminology:

| Step               | Description                     | AM (Account Manager) | CO (Compliance Officer) | FC (Finance Clerk) | IT (IT Support) |
|---------------------|---------------------------------|----------------------|------------------------|-------------------|-----------------|
| Receive Application | Handled by the **AM**            | R (Responsible)      |                        |                   |                 |
| KYC Review          | Performed by the **CO**         |                      | R (Responsible)       |                   |                 |
| Order Vetting       | The **AM** checks data          | R (Responsible)      |                        |                   |                 |
| Payment Clearance   | The **FC** initiates settlement |                      |                        | R (Responsible)   |                 |
| Record Update       | **IT** refreshes the CRM        |                      |                        |                   | R (Responsible) |
| Notify Customer     | The **AM** emails confirmation  | R (Responsible)      |                        |                   |                 |

### Key:
- **R**: Responsible (primary role for execution)
- All other columns are blank as no other roles are actively involved in each step.

The terms "KYC Review", "Order Vetting", and "Payment Clearance" strictly adhere to FinCo’s policies.
