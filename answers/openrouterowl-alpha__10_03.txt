

|                                    | Front-Office Analyst | Compliance Analyst | Compliance Officer | Treasury Ops |
|------------------------------------|:--------------------:|:------------------:|:------------------:|:------------:|
| **Intake Vetting**                 | R                    | C                  |                    | I            |
| **KYC Review**                     | I                    | R                  | C                  |              |
| **AML Screening**                  | I                    | R                  | C                  |              |
| **Risk Scoring**                   | I                    | R                  | C                  |              |
| **Compliance Officer Approval**    | I                    | C                  | R                  |              |
| **Funds Release**                  | I                    |                    |                    | R            |
| **Notification to Originating Analyst** | I               |                    |                    | R            |

**Legend:** R = Responsible · A = Accountable · C = Consulted · I = Informed

**Notes:**

- Intake Vetting is **Responsible** by the Front-Office Analyst, with the **Compliance Analyst** Consulted and **Treasury Ops** Informed of the completed outcome.
- KYC Review and AML Screening are both **Responsible** by the **Compliance Analyst**, with the **Compliance Officer** Consulted throughout and the **Front-Office Analyst** Informed.
- Risk Scoring is **Responsible** by the **Compliance Analyst** (acts may be **Accountable** for correctness), with the **Compliance Officer** Consulted and the **Front-Office Analyst** Informed.
- For approvals when the Risk Scoring result exceeds 70, the **Compliance Officer** is **Responsible** for granting approval, with the **Compliance Analyst** Consulted and the **Front-Office Analyst** Informed.
- Treasury Ops is **Responsible** for Funds Release and for delivering the final Notification to the Originating Analyst (the one performed the Intake Vetting).
