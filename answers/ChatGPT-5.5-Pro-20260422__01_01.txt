### Decision points in Contoso’s refund process

| Decision point               | Condition / check                                                                                                                 | Outcome                                                                                                                                                 |
| ---------------------------- | --------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Eligibility validation**   | The triage agent checks the claim against the **FlightManifest** and **WeatherReport**.                                           | Only eligible claims proceed to finance pre-approval.                                                                                                   |
| **Finance pre-approval**     | The finance controller signs off using the **EligibilityChecklist**.                                                              | **Process_Refund_Payment** may not begin unless **Finance_Controller_PreApprove** has succeeded.                                                        |
| **High-value refund branch** | Refund amount is **greater than €1,000**.                                                                                         | If yes, the case must go through **Senior_Auditor_Review** and include the **AuditTrail** before payment. If no, senior auditor review is not required. |
| **Payment readiness gate**   | Payment requires successful finance pre-approval, any required senior auditor review, and a signed **BankTransferAuthorization**. | The finance team may execute **Process_Refund_Payment** only when these conditions are satisfied.                                                       |

### Required documents and artefacts

| Document / artefact           | Used in / required for                                                     |
| ----------------------------- | -------------------------------------------------------------------------- |
| **RefundRequestForm**         | Submitted by the customer in **Customer_Submit_Claim**.                    |
| **FlightManifest**            | Used by **Triage_Agent_Validate_Eligibility**.                             |
| **WeatherReport**             | Used by **Triage_Agent_Validate_Eligibility**.                             |
| **EligibilityChecklist**      | Used by **Finance_Controller_PreApprove**.                                 |
| **AuditTrail**                | Required for **Senior_Auditor_Review** on refunds greater than **€1,000**. |
| **BankTransferAuthorization** | Must be signed before **Process_Refund_Payment**.                          |
| **RefundConfirmationPDF**     | Emailed to the passenger in **Issue_Confirmation_Letter**.                 |
