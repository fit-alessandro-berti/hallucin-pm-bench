**Question 1.** Yes, it is possible for T5 to fire before T3.

**Question 2.** Minimal firing sequence: **T1 → T2 → T7 → T4 → T5**.

**Explanation:**  
T5 requires both P2 and P5. P5 is produced by T4, which needs P4. P4 can be generated by T7 (via T2) instead of T3. T3 consumes P2, so using it would remove the token needed for T5. The sequence T1, T2, T7, T4, T5 produces P6 without ever firing T3, thus T5 fires before T3.
