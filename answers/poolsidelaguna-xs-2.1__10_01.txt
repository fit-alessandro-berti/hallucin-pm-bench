**RACI Matrix for Order-to-Cash Workflow**  

| Activity               | Account Manager (AM) | Compliance Officer (CO) | Finance Clerk (FC) | IT Support (IT) |  
|------------------------|----------------------|-------------------------|--------------------|-----------------|  
| **Receive Application** | **R/A**              |                         |                    |                 |  
| **KYC Review**          |                      | **R/A**                 |                    |                 |  
| **Order Vetting**       | **R/A**              |                         |                    |                 |  
| **Payment Clearance**   |                      |                         | **R/A**            | **I**           |  
| **Record Update**       | **I**                |                         | **I**              | **R/A**         |  
| **Notify Customer**     | **R/A**              | **I**                   | **I**              | **I**           |  

### Key:  
- **R** = Responsible (performs the work)  
- **A** = Accountable (owns the outcome)  
- **C** = Consulted (provides input)  
- **I** = Informed (kept updated on progress/results)  

### Notes:  
1. **Receive Application**: AM is solely responsible and accountable.  
2. **KYC Review**: CO handles verification; AM may be informed post-completion.  
3. **Order Vetting**: AM ensures data completeness; no cross-role dependencies.  
4. **Payment Clearance**: FC initiates settlement; IT updates CRM post-clearance.  
5. **Record Update**: IT maintains CRM; AM and FC are informed upon completion.  
6. **Notify Customer**: AM sends confirmation; all roles are informed of the outcome.  

This matrix strictly adheres to the preferred terminology rules and reflects role responsibilities per the process description.
