# RACI Matrix: AtlasRemit Cross-Border Payment Process

| Task ID | Task Description | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :--- | :---: | :---: | :---: |
| 1 | Receive customer transfer instruction and log it in the system. | **R/A** | I | |
| 2 | Screen the payment against the sanctions list. | **R/A** | I | |
| 3 | Perform the KYC Review on both sender and beneficiary. | C | **R/A** | |
| 4 | Approve the transaction if flagged as high-risk. | I | **R/A** | |
| 5 | Release the payment upon approval; ensure SWIFT message is sent. | **R** | I | **A** |
| 6 | Archive the case file and notify the Regulatory Liaison. | I | I | **R/A** |

**Key:**  
**R** = Responsible (performs the work)  
**A** = Accountable (ultimately answerable)  
**C** = Consulted (provides input)  
**I** = Informed (kept up-to-date)
