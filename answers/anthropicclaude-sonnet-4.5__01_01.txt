# Contoso Airlines Flight-Refund Process Summary

## Decision Points

1. **High-value threshold decision** (after Finance_Controller_PreApprove)
   - **Condition**: Refund amount > €1,000
   - **Outcome**: 
     - If YES → route to Senior_Auditor_Review
     - If NO → proceed directly to Process_Refund_Payment

## Required Documents by Activity

| Activity | Required Document(s) |
|----------|---------------------|
| Customer_Submit_Claim | *RefundRequestForm* |
| Triage_Agent_Validate_Eligibility | *FlightManifest*, *WeatherReport* |
| Finance_Controller_PreApprove | *EligibilityChecklist* |
| Senior_Auditor_Review | *AuditTrail* |
| Process_Refund_Payment | *BankTransferAuthorization* (signed) |
| Issue_Confirmation_Letter | *RefundConfirmationPDF* |
