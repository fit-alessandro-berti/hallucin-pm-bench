**RACI Matrix: FinCo Order-to-Cash Workflow**

| Activity / Role       | Account Manager (AM) | Compliance Officer (CO) | Finance Clerk (FC) | IT Support (IT) |
| :-------------------- | :------------------: | :---------------------: | :---------------: | :-------------: |
| 1. Receive Application |         A/R          |            -            |         -         |        -        |
| 2. KYC Review         |          C           |           A/R           |         -         |        -        |
| 3. Order Vetting      |         A/R          |            I            |         -         |        -        |
| 4. Payment Clearance  |          A           |            -            |        R          |        -        |
| 5. Record Update      |          I           |            -            |         -         |       R         |
| 6. Notify Customer    |         A/R          |            -            |         -         |        -        |

**Legend:**  
**R** = Responsible (executes the task)  
**A** = Accountable (owns the task, approves work)  
**C** = Consulted (two-way communication)  
**I** = Informed (one-way notification)  
**-** = No involvement
