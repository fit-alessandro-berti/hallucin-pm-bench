Below is the RACI for AtlasRemit cross-border payment process using the enforced vocabulary:

| Task | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :---: | :---: | :---: |
| 1. Receive customer transfer instruction and log in system | **R, A** | I | - |
| 2. Screen payment against sanctions list | **R** | **A, C** | - |
| 3. Perform **KYC Review** on sender and beneficiary | C | **R, A** | - |
| 4. Approve high-risk transaction | I | **R, A** | - |
| 5. Release payment and ensure SWIFT message transmission | **R** | **A** | **R** |
| 6. Archive case file and notify stakeholders | I | I | **R, A** |

**Legend:** R = Responsible, A = Accountable, C = Consulted, I = Informed
