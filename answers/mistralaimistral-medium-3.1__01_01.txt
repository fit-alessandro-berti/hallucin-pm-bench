### **Decision Points in Contoso Airlines’ Flight-Refund Process**
1. **Eligibility Check (Triage_Agent_Validate_Eligibility)**
   - **Decision**: Is the claim valid based on the *FlightManifest* and *WeatherReport*?
   - **Outcome**:
     - If **invalid** → Process terminates (no further steps).
     - If **valid** → Proceeds to **Finance_Controller_PreApprove**.

2. **High-Value Threshold Check (after Finance_Controller_PreApprove)**
   - **Decision**: Is the refund amount **> €1,000**?
   - **Outcome**:
     - If **yes** → Route to **Senior_Auditor_Review** (requires *AuditTrail*).
     - If **no** → Skip audit; proceed directly to **Process_Refund_Payment**.

3. **Senior Auditor Approval (High-Value Branch Only)**
   - **Decision**: Does the *AuditTrail* meet compliance standards?
   - **Outcome**:
     - If **approved** → Proceed to **Process_Refund_Payment**.
     - If **rejected** → Process terminates (no refund).

4. **Payment Authorization Check (Process_Refund_Payment)**
   - **Decision**: Is the *BankTransferAuthorization* signed and present?
   - **Outcome**:
     - If **yes** → Execute payment.
     - If **no** → Payment cannot proceed (process stalls until resolved).

### **Required Documents**
| **Activity**                     | **Mandatory Documents**                          |
|-----------------------------------|--------------------------------------------------|
| **Customer_Submit_Claim**         | *RefundRequestForm*                              |
| **Triage_Agent_Validate_Eligibility** | *FlightManifest*, *WeatherReport*           |
| **Finance_Controller_PreApprove** | *EligibilityChecklist*                           |
| **Senior_Auditor_Review**         | *AuditTrail* (high-value claims only)            |
| **Process_Refund_Payment**        | *BankTransferAuthorization* (signed)             |
| **Issue_Confirmation_Letter**     | *RefundConfirmationPDF*                         |

---
**Note**: No other decision points or documents exist in this process. The legacy "CustomerService_Agent_Decide" activity is **not part** of Contoso’s workflow.
