### RACI Matrix – FinCo Order-to-Cash Process (Preferred Terminology)

| **Activity**               | **Account Manager (AM)** | **Compliance Officer (CO)** | **Finance Clerk (FC)** | **IT Support (IT)** |
|----------------------------|--------------------------|-----------------------------|------------------------|---------------------|
| Receive Application        | **R, A**                 | I                           |                        |                     |
| KYC Review                 | C                        | **R, A**                    |                        |                     |
| Order Vetting              | **R, A**                 | C                           |                        |                     |
| Payment Clearance          | C                        |                             | **R, A**               |                     |
| Record Update              | I                        |                             |                        | **R, A**            |
| Notify Customer            | **R, A**                 | I                           | I                      |                     |

**Legend:**  
**R** = Responsible (does the work)  
**A** = Accountable (final authority/approval)  
**C** = Consulted (provides input)  
**I** = Informed (kept up-to-date)
