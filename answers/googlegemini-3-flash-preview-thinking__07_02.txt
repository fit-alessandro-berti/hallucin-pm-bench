Based on a comparison of the BPMN XML for the *Loan_Process*, the following changes have been identified:

**Added tasks**
*   **Verify Identity** (`Task_VerifyIdentity`): A new preceding step before the application review.
*   **Request Additional Documents** (`Task_RequestDocs`): A new task following the decision gateway.
*   **Final Reject** (`Task_FinalReject`): This task replaces the previous rejection task with a new identifier and path.

**Removed tasks**
*   **Reject Loan** (`Task_RejectLoan`): This specific task ID and element from v1 have been removed.

**Gateway-type changes**
*   **Gateway_Decision**: The element type was changed from an **Exclusive Gateway** (`exclusiveGateway`) to an **Inclusive Gateway** (`inclusiveGateway`), allowing for multiple concurrent paths (e.g., approving while also requesting documents).
