Here’s the RACI matrix for AtlasRemit’s cross-border payment process, with all terms updated to the mandated vocabulary.

**Key**: R = Responsible, A = Accountable, C = Consulted, I = Informed

| Task | Transaction Analyst | Regulatory Liaison |
| --- | --- | --- |
| 1. Receive customer transfer instruction and log it in the system | R, A | I |
| 2. Screen the payment against the sanctions list | R, A | C |
| 3. Perform **KYC Review** on both sender and beneficiary | C | R, A |
| 4. Approve transaction if flagged as high-risk | I | R, A |
| 5. Release the payment and ensure the SWIFT message is sent | R, A | I |
| 6. Archive the case file | R, A | I |

**Notes on vocabulary mapping applied**:
- "Payment clerk" → **Transaction Analyst** 
- "Compliance officer" → **Regulatory Liaison**
- "Customer due diligence" → **KYC Review**

IT Support responsibilities from the source text have been assigned to **Transaction Analyst** to comply with the mandate to use only the three specified roles.
