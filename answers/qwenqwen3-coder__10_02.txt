# RACI Matrix for AtlasRemit Cross-Border Payment Process

| Process Activity | Transaction Analyst | Regulatory Liaison | IT Support |
|------------------|---------------------|-------------------|------------|
| Receive customer transfer instruction and log in system | **R** | I | I |
| Screen payment against sanctions list | **R** | C | I |
| Conduct KYC Review on sender and beneficiary | C | **R** | I |
| Approve high-risk transactions | C | **A** | I |
| Release payment upon approval | **R** | C | I |
| Ensure SWIFT message transmission | I | I | **R** |
| Archive case file | I | I | **R** |
| Notify Regulatory Liaison of archived case | I | **I** | R |

**Key:**
- **R** = Responsible (performs the work)
- **A** = Accountable (ultimately answerable for completion)
- **C** = Consulted (provides input/expertise)
- **I** = Informed (kept updated on progress/completion)
