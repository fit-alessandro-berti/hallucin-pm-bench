# Summary of Changes: Loan_Process (v1 → v2)

## Added tasks
- **Task_VerifyIdentity** ("Verify Identity") – inserted between StartEvent_1 and Task_ReviewApplication
- **Task_RequestDocs** ("Request Additional Documents") – new branch from Gateway_Decision

## Removed tasks
- **Task_RejectLoan** ("Reject Loan") – replaced functionally by Task_FinalReject (new task, not a rename, since gateway logic and branching structure changed)

*Note: Task_ApproveLoan is retained (relabeled "Auto-Approve Loan"), which is a cosmetic rename and not counted as added/removed per instructions.*

## Gateway-type changes
- **Gateway_Decision**: changed from **Exclusive Gateway** (v1) → **Inclusive Gateway** (v2)
  - This changes the branching semantics from mutually exclusive single-path selection to potential multi-path (conditional OR) execution, and the gateway now supports three outgoing branches instead of two.
