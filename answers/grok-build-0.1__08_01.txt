**Yes**, it is possible.

**Firing sequence**: T1, T2, T5.

- Start: (p0:1).
- T1 fires → (p1:1, p2:1).
- T2 fires → (p2:1, p3:1). (p4 still = 0.)
- At this point T5 is enabled (p3 ≥ 1 and p4 = 0), so T5 can fire, consuming the token from p3 and producing one in p6. T3 has not fired in this sequence.

This is reachable because T3 is the only transition that can ever place a token in p4; by firing T2 before T3 we obtain the token in p3 while the inhibitor condition for T5 remains satisfied.
