Based on my analysis of the two BPMN XML files for Acme Bank's *Loan_Process*, here are all changes:

### Added tasks
- **Verify Identity** (id: `Task_VerifyIdentity`) – New initial task after start event
- **Request Additional Documents** (id: `Task_RequestDocs`) – New intermediate task
- **Final Reject** (id: `Task_FinalReject`) – New rejection task

### Removed tasks
- **Reject Loan** (id: `Task_RejectLoan`) – Removed and replaced by `Task_FinalReject`

### Gateway-type changes
- **Gateway_Decision** changed from **Exclusive Gateway** (one-out-of-many) to **Inclusive Gateway** (multiple-outcome)
  - v1: Exclusive (binary yes/no decision)
  - v2: Inclusive (three-outcome: eligible, missing docs, ineligible)

**Note:** The approve task name changed from "Approve Loan" to "Auto-Approve Loan" (cosmetic, ignored as instructed).
