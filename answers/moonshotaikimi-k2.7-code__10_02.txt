 ## AtlasRemit Cross-Border Payment Process — RACI Matrix

| Task / Activity | Transaction Analyst | Regulatory Liaison | IT Support |
|---|---|---|---|
| 1. Receive transfer instruction and log in system | **R / A** | I | I |
| 2. Screen payment against sanctions list | **R** | **A** | I |
| 3. Conduct KYC Review on sender and beneficiary | C | **R / A** | I |
| 4. Approve high-risk transaction | I | **R / A** | I |
| 5. Release payment | **R / A** | I | I |
| 6. Ensure SWIFT message is sent | I | I | **R / A** |
| 7. Archive case file | I | I | **R / A** |
| 8. Notify Regulatory Liaison upon case closure | I | I | **R / A** |

**RACI key**

- **R = Responsible**: Performs the actual work for the task.  
- **A = Accountable**: Ultimately answerable for correct completion; each row has one **A**.  
- **C = Consulted**: Provides input or information before the task proceeds.  
- **I = Informed**: Kept up to date on the outcome of the task.
