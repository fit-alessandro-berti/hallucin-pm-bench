Here is the RACI matrix for FinCo’s order-to-cash workflow.

**Legend**
* **R** = Responsible (performs the work)
* **A** = Accountable (owns the outcome; exactly one per row)
* **C** = Consulted (provides input)
* **I** = Informed (kept up to date)
* **-** = Not involved

**RACI Matrix**

| Activity | Account Manager | Compliance Officer | Finance Clerk | IT Support |
| :--- | :---: | :---: | :---: | :---: |
| Receive Application | R/A | I | - | - |
| KYC Review | I | R/A | - | - |
| Order Vetting | R/A | C | I | - |
| Payment Clearance | I | - | R/A | - |
| Record Update | A | - | I | R |
| Notify Customer | R/A | - | I | - |
