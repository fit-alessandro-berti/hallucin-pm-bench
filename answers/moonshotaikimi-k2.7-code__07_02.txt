### Added tasks
- **Verify Identity** (`Task_VerifyIdentity`) — inserted between the start event and *Review Application*.
- **Request Additional Documents** (`Task_RequestDocs`) — new outgoing branch from the decision gateway.
- **Final Reject** (`Task_FinalReject`) — replaces the previous *Reject Loan* path.

### Removed tasks
- **Reject Loan** (`Task_RejectLoan`) — removed; its path is replaced by *Final Reject*.

### Gateway-type changes
- **Decision gateway changed from Exclusive to Inclusive**  
  - v1: `exclusiveGateway` named “Eligible?” with two branches (`yes` / `no`).  
  - v2: `inclusiveGateway` named “Eligibility Check” with three branches (`eligible` / `missing docs` / `ineligible`).
