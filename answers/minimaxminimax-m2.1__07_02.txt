# BPMN Process Comparison Summary: Loan_Process

## Added tasks
- **Task_VerifyIdentity** – New initial task after Start event
- **Task_RequestDocs** – New task for handling missing documentation
- **Task_FinalReject** – New task for ineligible cases

## Removed tasks
- **Task_RejectLoan** – Removed; replaced by a split path (Request Additional Documents / Final Reject)

## Gateway-type changes
- **Gateway_Decision**: Changed from **exclusiveGateway** to **inclusiveGateway**
- The single binary decision ("yes" → Approve, "no" → Reject) now has **three outgoing paths**: "eligible" → Auto-Approve Loan, "missing docs" → Request Additional Documents, "ineligible" → Final Reject
