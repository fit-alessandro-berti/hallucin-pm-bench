### AtlasRemit Process Management Documentation: Transfer Workflow

The following RACI matrix outlines the responsibilities for the standard cross-border transfer workflow. 

| Task / Activity | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :---: | :---: | :---: |
| Receive transfer instruction and log in system | **R** | **I** | - |
| Screen payment against sanctions list | **R** | **C** | - |
| Perform **KYC Review** (Sender & Beneficiary) | **I** | **R/A** | - |
| High-risk transaction approval | - | **R/A** | - |
| Release approved payment | **R** | **I** | - |
| Transmission of SWIFT message | - | - | **R/A** |
| Archive case file | - | - | **R** |
| Case file completion notification | - | **I** | **R** |

***

**Legend:**
*   **R: Responsible** – The person who performs the work to complete the task.
*   **A: Accountable** – The person ultimately answerable for the correct and thorough completion of the task.
*   **C: Consulted** – Those whose opinions are sought; two-way communication.
*   **I: Informed** – Those who are kept up-to-date on progress; one-way communication.
