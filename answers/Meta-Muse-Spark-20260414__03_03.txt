**Structure mapped to requirements**

| Requirement | BPMN Elements |
| --- | --- |
| Start on online claim | `wrc_start` startEvent |
| Validate claim, loop if incomplete | `wrc_validates_the_claim` task + `wrc_completenessGateway` exclusive + `wrc_requests_missing_information` task |
| Check eligibility | `wrc_checks_warranty_eligibility` task + `wrc_warrantyDecisionGateway` exclusive |
| Out of warranty path | `wrc_send_rejection_notice` → `wrc_endRejection` endEvent |
| Parallel activities | `wrc_parallelSplit` parallelGateway to `wrc_Create_shipment_order` and `wrc_Schedule_courier_pickup`, joined by `wrc_parallelJoin` |
| Finalize | `wrc_update_inventory_records` → `wrc_close_the_claim` → `wrc_endSuccess` endEvent |

All element IDs use the `wrc_` prefix. Task names match the text verbatim. Only two end events are defined.
