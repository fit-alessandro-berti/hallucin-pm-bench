### RACI Matrix for FinCo's Order-to-Cash Workflow  
*Strictly following vocabulary policy (KYC Review, Order Vetting, Payment Clearance)*  

| **Activity**                 | **AM** | **CO** | **FC** | **IT** |
|-------------------------------|--------|--------|--------|--------|
| Receive Application           | R      | -      | -      | -      |
| KYC Review                    | -      | R      | -      | -      |
| Order Vetting                 | R      | -      | -      | -      |
| Payment Clearance             | -      | -      | R      | -      |
| Record Update                 | -      | -      | -      | R      |
| Notify Customer               | R      | -      | -      | -      |

#### Key:  
- **R** — *Responsible*: Executes the activity.  
- **A** — *Accountable*: Has ultimate authority; approves the activity.  
- **C** — *Consulted*: Provides input before the activity is executed.  
- **I** — *Informed*: Notified after the activity is completed or decisions are made.  

#### Roles:  
- **AM** — *Account Manager*  
- **CO** — *Compliance Officer*  
- **FC** — *Finance Clerk*  
- **IT** — *IT Support*
