# AtlasRemit Cross-Border Payment Process RACI Matrix

| Process Step | Transaction Analyst | Regulatory Liaison | IT Support |
|-------------|:-------------------:|:------------------:|:----------:|
| Receive and log transfer instruction | R/A | I | - |
| Screen payment against sanctions list | R/A | C | - |
| Perform KYC Review on sender and beneficiary | C | R/A | - |
| Approve high-risk transactions | I | R/A | - |
| Release payment | R/A | I | C |
| Send SWIFT message | C | I | R/A |
| Archive case file | - | I | R/A |

**Legend:**
- R: Responsible (performs the work)
- A: Accountable (ultimately answerable)
- C: Consulted (provides input)
- I: Informed (kept updated)
