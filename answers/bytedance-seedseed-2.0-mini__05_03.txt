### Added/Changed XML Elements
```xml
<!-- Interrupting boundary timer for SLA enforcement on Create Account task -->
<bpmn:boundaryEvent id="BoundaryTimer_SLA" cancelActivity="true" attachedToRef="Activity_CreateAccount">
  <bpmn:timerEventDefinition>
    <bpmn:timeDuration xsi:type="bpmn:tFormalExpression">P48H</bpmn:timeDuration>
  </bpmn:timerEventDefinition>
</bpmn:boundaryEvent>
<!-- Escalation flow for SLA breaches to existing unused review task -->
<bpmn:sequenceFlow id="Flow_SLA_Escalation" sourceRef="BoundaryTimer_SLA" targetRef="Activity_AMReview"/>
```

### Justification
This adds an interrupting 48-hour boundary timer tied directly to the "Create Account in Core System" task that triggers exactly 48 hours after the task starts (aligning with the SLA deadline from the end of identity verification), routing SLA breaches to the pre-existing unused Account Manager Review task without adding any new activities or subprocesses.
