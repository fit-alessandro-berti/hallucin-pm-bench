# NimbusPay Wire-Transfer Process – RACI Matrix

| Activity | Front-Office Analyst | Compliance Analyst | Compliance Officer | Treasury Ops |
|---|---|---|---|---|
| Intake Vetting | **R** | | | **I** |
| KYC Review | **C** | **R** | | **I** |
| AML Screening | **C** | **R** | | **I** |
| Risk Scoring | **C** | **R** | **A** | **I** |
| Approval (Risk ≤ 70) | | | | **R, A** |
| Approval (Risk > 70) | | **C** | **R, A** | **I** |
| Fund Release | | **I** | **C** | **R, A** |
| Notification to Originating Analyst | | | **C** | **R, A** |

---

**Key:**
- **R** = Responsible (executes the work)
- **A** = Accountable (final authority; owns outcome)
- **C** = Consulted (provides input; two-way dialogue)
- **I** = Informed (kept in loop; one-way communication)
