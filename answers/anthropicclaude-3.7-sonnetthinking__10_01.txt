# FinCo Order-to-Cash RACI Matrix

| Activity | Account Manager (AM) | Compliance Officer (CO) | Finance Clerk (FC) | IT Support (IT) |
|----------|----------------------|-------------------------|--------------------|-----------------| 
| Receive Application | RA | I | I | - |
| KYC Review | C | RA | - | - |
| Order Vetting | RA | C | - | - |
| Payment Clearance | I | - | RA | - |
| Record Update | I | I | I | RA |
| Notify Customer | RA | - | - | - |

**Legend:**
- R: Responsible (performs the work)
- A: Accountable (ultimately answerable)
- C: Consulted (provides input)
- I: Informed (kept updated)

This matrix adheres to the proper terminology for all process steps while maintaining clear role assignments across the entire workflow.
