### RACI Matrix for Wire-Transfer Process  
Using **strictly authorized vocabulary** as per policy:  

| Activity          | Front-Office Analyst | Compliance Analyst | Compliance Officer | Treasury Ops |  
|-------------------|----------------------|--------------------|--------------------|--------------|  
| **Intake Vetting** | R, A                |                    |                    |              |  
| **KYC Review**     |                     | R, A               |                    |              |  
| **AML Screening**  |                     | R, A               |                    |              |  
| **Risk Scoring**   |                     | R, A               |                    |              |  
| **Approval**<sup>†</sup> |                     | C                  | R, A               |              |  
| **Schedule Payment**<sup>‡</sup> |              |                    |                    | R, A         |  
| **Release Funds**  | I                   |                    |                    | R, A         |  

---

### Key:  
- **R** = Responsible (executes the activity)  
- **A** = Accountable (owns outcome/approval)  
- **C** = Consulted (provides input pre-decision)  
- **I** = Informed (notified post-activity)  

<sup>†</sup> *Approval*: Triggered only if **Risk Scoring** >70.  
<sup>‡</sup> *Schedule Payment*: Triggered only if **Risk Scoring** ≤70.  

### Notes:  
- **Intake Vetting**, **KYC Review**, **AML Screening**, and **Risk Scoring** retain exact mandated terminology.  
- **Approval**, **Schedule Payment**, and **Release Funds** reflect explicit actions from the narrative (no synonyms added).  
- **Compliance Analyst** is **Consulted** during *Approval* to provide context for escalation.  
- **Front-Office Analyst** is **Informed** when funds are released (per narrative: "notifies the originating analyst").
