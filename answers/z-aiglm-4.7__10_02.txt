| Task / Activity | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :---: | :---: | :---: |
| Receive and log transfer instruction | **R** | | |

| Ensure SWIFT message transmission | | | **R** |
| Archive case file | | **I** | **R** |

**Legend:**
*   **R (Responsible):** The individual who performs the work.
*   **A (Accountable):** The individual ultimately answerable for the activity or decision.
*   **I (Informed):** The individual who needs to be kept up-to-date on progress or completion.
