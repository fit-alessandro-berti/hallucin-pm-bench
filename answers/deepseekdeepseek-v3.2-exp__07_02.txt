Based on a comparison of the two BPMN XML versions, here is a summary of all changes, grouped under the required headings.

### Added tasks
- **Task_VerifyIdentity** ("Verify Identity") has been added between the start event and "Review Application".
- **Task_RequestDocs** ("Request Additional Documents") has been added as a new outgoing path from the gateway.
- **Task_FinalReject** ("Final Reject") has been added as a new outgoing path from the gateway, replacing the previous "Reject Loan" task.

### Removed tasks
- **Task_RejectLoan** ("Reject Loan") has been removed.

### Gateway-type changes
- The gateway **Gateway_Decision** has been changed from an **Exclusive Gateway** to an **Inclusive Gateway**. Its name has also been changed from "Eligible?" to "Eligibility Check".
