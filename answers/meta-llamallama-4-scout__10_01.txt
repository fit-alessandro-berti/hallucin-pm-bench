## RACI Matrix for Order-to-Cash Workflow

### Activities and Responsible Roles

| Activity                | Account Manager (AM) | Compliance Officer (CO) | Finance Clerk (FC) | IT Support (IT) |
|-------------------------|----------------------|-------------------------|--------------------|-----------------|
| Receive Application     | R                   |                         |                    |                 |
| KYC Review              |                    | R                       |                    |                 |
| Order Vetting           | R                   |                         |                    |                 |
| Payment Clearance       |                    |                         | R                  |                 |
| Record Update           |                    |                         |                    | R               |
| Notify Customer         | A                   |                         |                    |                 |

### RACI Legend
- **R**: Responsible - The person who performs the work.
- **A**: Accountable - The person who delegates the work. Ensures the task is completed.
- **C**: Consulted - The person who provides input.
- **I**: Informed - The person who is kept up-to-date on the progress.

### Notes
- In the **KYC Review** step, the Compliance Officer is solely responsible.
- The **Account Manager** is accountable for **Notify Customer** but also performs **Receive Application**, **Order Vetting**, and notifies the customer.
- The **Finance Clerk** is responsible for **Payment Clearance**.
- **IT Support** is responsible for **Record Update**.
