**Decision Points in Acme's Micro-loan Fast-Track Process**

1. **Quick KYC Verification (KYC)**  
   Confirms the applicant's ID card and mobile phone. If verification fails, the application cannot advance.

2. **Underwriter Alignment Gate**  
   After dual underwriters assign independent risk scores, this gate checks if the scores differ by ≤ 2 points. If the gap is larger, the application is escalated to the Harmonisation Committee for a tie-break decision.

3. **Amount Threshold Decision**  
   Evaluates the loan amount:  
   - If **< €15 000**, the system auto-approves.  
   - If **≥ €15 000**, the process proceeds to the Micro-loan Board.

4. **Final Micro-loan Board Approval (MBA)**  
   For loans ≥ €15 000, the board votes to approve or reject the application.

**Every Required Document at Submission Time**

- Digital Application Form (ML-APP-01)  
- Proof of Address (≤ 3 months old)  
- Latest Income Statement (single pay-stub or tax slip)  
- Community Impact Statement (CIS) template  
- Dual Underwriter Score-Sheet  
- Neighbourhood Feedback Form (NFF)
