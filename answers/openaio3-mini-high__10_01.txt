Below is a RACI matrix for FinCo’s order-to-cash workflow. Note that the steps have been relabeled using the approved vocabulary. Each activity is mapped against the roles: Account Manager (AM), Compliance Officer (CO), Finance Clerk (FC), and IT Support (IT).

────────────────────────────────────────────────────────────
Activity                | AM        | CO        | FC        | IT
────────────────────────────────────────────────────────────
1. Receive Application  | A, R      |           |           |  
────────────────────────────────────────────────────────────
2. KYC Review           | I         | A, R      |           |  
────────────────────────────────────────────────────────────
3. Order Vetting        | A, R      |           |           |  
────────────────────────────────────────────────────────────
4. Payment Clearance    | I         |           | A, R      |  
────────────────────────────────────────────────────────────
5. Record Update        |           |           |           | A, R
────────────────────────────────────────────────────────────
6. Notify Customer      | A, R      |           |           |  
────────────────────────────────────────────────────────────

Legend:
 • A = Accountable (owns the process step)
 • R = Responsible (executes the work)
 • I = Informed (kept in the loop)

In this mapping:
• The Account Manager (AM) handles the application receipt, order vetting, and customer notification.
• The Compliance Officer (CO) conducts the KYC Review.
• The Finance Clerk (FC) carries out the Payment Clearance.
• IT Support (IT) manages the Record Update. 

This matrix strictly uses the approved terms and provides clear role ownership for each step in the workflow.
