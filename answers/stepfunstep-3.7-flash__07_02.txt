### Added tasks
- Verify Identity (ID: `Task_VerifyIdentity`): New task inserted between the start event and the pre-existing Review Application step.
- Request Additional Documents (ID: `Task_RequestDocs`): New task added as an additional outgoing path from the decision gateway, for cases where applicant documentation is missing.
- Final Reject (ID: `Task_FinalReject`): New task added as an outgoing path from the decision gateway for ineligible applicants.

### Removed tasks
- Reject Loan (ID: `Task_RejectLoan`): Original task for rejecting ineligible loan applications, removed from the v2 process.

### Gateway-type changes
- The decision gateway (ID: `Gateway_Decision`) was changed from an Exclusive Gateway to an Inclusive Gateway, enabling multiple parallel process paths to be triggered instead of only one mutually exclusive path.
