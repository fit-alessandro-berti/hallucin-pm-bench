# Decision Points

1. **Underwriter Alignment Gate** – compares the two independent risk scores:
   - If the scores from **Senior Underwriter A** and **Shadow Underwriter B** differ by **≤ 2 points** → continue to the **Neighbourhood Feedback Check (NFC)**.
   - Else → escalate to the **Harmonisation Committee** for a tie-break decision, then continue.

2. **Amount Threshold Decision** – branches on the loan amount:
   - If **amount < €15 000** → the system auto-approves, moving straight to **Disburse Funds**.
   - If **amount ≥ €15 000** → proceed to **Final Micro-loan Board Approval (MBA)**.

3. **Final Micro-loan Board Approval (MBA)** – the board votes:
   - **Approve** → **Disburse Funds**.
   - **Reject** → the application does not proceed.

# Required Documents (at submission time)

- **Digital Application Form (ML-APP-01)**
- **Proof of Address** (≤ 3 months old)
- **Latest Income Statement** (single pay-stub or tax slip)
- **Community Impact Statement (CIS)** template
- **Dual Underwriter Score-Sheet**
- **Neighbourhood Feedback Form (NFF)**
