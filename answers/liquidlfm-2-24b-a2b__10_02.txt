### RACI Matrix for AtlasRemit Process

| Activity / Task                                                                 | Responsible | Accountable | Consulted | Informed |
|---------------------------------------------------------------------------------|-------------|-------------|-----------|----------|
| Log payment instruction in the system                                           | Clerk       | Manager     |           |          |
| Screen payment against sanctions list                                          | Clerk       | Manager     |           |          |
| Perform **KYC Review** on sender and beneficiary                               | Compliance Officer | Manager |           |          |
| Approve high‑risk transaction if flagged                                       | Compliance Officer | Manager |           |          |
| Release payment after approval                                                 | Clerk       | Manager     |           |          |
| Ensure SWIFT message is sent (IT Support)                                       | IT Support  | Manager     |           |          |
| Archive case file                                                              | IT Support  | Manager     |           |          |
| Notify compliance officer of archived case                                     | IT Support  | Manager     |           |          |

**Notes:**  
- “Payment clerk” and “compliance officer” are used as the primary operational roles.  
- “Manager” represents the Accountable authority (e.g., Process Lead).  
- “IT Support” handles technical delivery tasks.  
- All tasks are covered; no role is left without a defined RACI label.
