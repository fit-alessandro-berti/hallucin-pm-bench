Below is the RACI matrix for the AtlasRemit cross-border payment process, structured strictly in accordance with your terminology requirements.

**Legend**  
**R** = Responsible (does the work)  
**A** = Accountable (owns the outcome/decision)  
**C** = Consulted (provides input)  
**I** = Informed (kept up to date)  

| Process Step | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :---: | :---: | :---: |
| Receive and log customer's transfer instruction | R / A | I | - |
| Screen payment against sanctions list | R | A | - |
| Perform KYC Review on sender and beneficiary | I | R / A | - |
| Approve high-risk transaction | I | R / A | - |
| Release payment | R | A | - |
| Ensure SWIFT message is sent | I | I | R / A |
| Archive case file | I | A | R |
| Notify of case completion | - | I | R / A |
