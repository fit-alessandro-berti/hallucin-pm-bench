**Question 2.** Transition **T5** requires a token in both **P2** and **P5**. Transition **P2** is populated only when **T1** fires. Transition **P5** is populated only after **T4** fires, which requires a token in **P4**. **P4** is populated by either **T2-T7** (via **P1**) or **T3** (via **P2**). 

Because the sequence must reach **P5**, **T4** must fire, which necessitates an input token in **P4**. That token must arrive via either **T3** or **T7**. If **T3** has not fired, **T5** cannot satisfy the requirement for a token in **P5** unless **T3** has already occurred to feed the chain leading to **P5**. Thus, **T3** is a structural prerequisite.
