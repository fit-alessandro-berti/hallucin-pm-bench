

### RACI Matrix for AtlasRemit Cross-Border Payment Process  

| Task                                                    | Responsible           | Accountable           | Consulted | Informed           |
|---------------------------------------------------------|-----------------------|-----------------------|-----------|--------------------|
| Receive transfer instruction and log in system          | Transaction Analyst   | Transaction Analyst   |           |                    |
| Screen payment against sanctions list                   | Transaction Analyst   | Transaction Analyst   |           |                    |
| Perform KYC Review on both sender and beneficiary       | Regulatory Liaison    | Regulatory Liaison    |           |                    |
| Approve transaction if flagged as high-risk             | Regulatory Liaison    | Regulatory Liaison    |           |                    |
| Release the payment                                     | Transaction Analyst   | Transaction Analyst   |           |                    |
| Ensure SWIFT message is sent                            | IT Support            | IT Support            |           |                    |
| Archive the case file                                   | IT Support            | IT Support            |           |                    |
| Notify Regulatory Liaison of completion                 | IT Support            | IT Support            |           | Regulatory Liaison |  

**Notes:**  
- **Responsible (R)**: Role performing the task.  
- **Accountable (A)**: Role with ultimate ownership of the task.  
- **Consulted (C)**: Role(s) providing input (not required in this process).  
- **Informed (I)**: Role(s) kept updated (only explicitly required for final notification).  
- All activities and roles strictly adhere to the mandated vocabulary.
