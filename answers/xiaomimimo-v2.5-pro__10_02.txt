# AtlasRemit Cross-Border Payment Process: RACI Matrix

Below is the RACI table for the described payment process, using only the mandated terminology for activities and roles.

| Task / Process Step                               | Transaction Analyst | Regulatory Liaison | IT Support | Notes & Mandated Terms Used |
| ------------------------------------------------- | :-----------------: | :----------------: | :--------: | --------------------------- |
| 1. Receive and log customer transfer instruction. | **R**               | -                  | -          | *“Transaction Analyst”* replaces “payment clerk”. |
| 2. Screen payment against sanctions list.         | **R**               | -                  | -          | Activity performed by Transaction Analyst. |
| 3. Perform **KYC Review** on sender & beneficiary. | **C**               | **R, A**           | -          | *“KYC Review”* mandated; role is *“Regulatory Liaison”*. |
| 4. Approve high-risk flagged transactions.        | **C**               | **R, A**           | -          | Approval authority resides with Regulatory Liaison. |
| 5. Release approved payment & send SWIFT message. | **R**               | **I**              | **R, A**   | Transaction Analyst releases; IT Support is **R** for SWIFT. |
| 6. Archive case file & notify Regulatory Liaison. | **I**               | **I**              | **R, A**   | IT Support archives and sends notification. |

### Legend & Key
*   **R – Responsible**: Performs the work to complete the task.
*   **A – Accountable**: Ultimately answerable for the task's completion and correctness.
*   **C – Consulted**: Provides input or expertise before or during the task.
*   **I – Informed**: Kept up-to-date on progress or outcomes.

### Mandated Terminology Applied
*   **Role**: **“Transaction Analyst”** (used for all references to the operational processor).
*   **Role**: **“Regulatory Liaison”** (used for all references to the compliance function).
*   **Activity**: **“KYC Review”** (used exclusively for customer due diligence activities).
