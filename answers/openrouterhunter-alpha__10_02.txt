# AtlasRemit Process RACI Matrix

**Vocabulary Key:**  
- **TA** = Transaction Analyst  
- **RL** = Regulatory Liaison  
- **IT** = IT Support  

| Task | TA (Transaction Analyst) | RL (Regulatory Liaison) | IT (IT Support) |
| :--- | :--- | :--- | :--- |
| 1. Receive and log customer transfer instruction | R | I | - |
| 2. Screen payment against sanctions list | R | I | - |
| 3. Perform KYC Review on sender and beneficiary | C | R, A | - |
| 4. Approve high-risk flagged transactions | I | R, A | - |
| 5. Release payment | R | I | - |
| 6. Send SWIFT message | C | - | R |
| 7. Archive case file and notify RL | I | I | R, A |

**Key:**  
- **R** = Responsible (executes the work)  
- **A** = Accountable (final authority, yes/no)  
- **C** = Consulted (provides input)  
- **I** = Informed (notified of outcome)
