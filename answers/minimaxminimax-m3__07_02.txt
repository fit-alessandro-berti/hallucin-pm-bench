### Summary of Changes

**Added tasks**
- `Task_VerifyIdentity` – "Verify Identity" (inserted between the start event and the review step).
- `Task_RequestDocs` – "Request Additional Documents" (new branch from the decision gateway for the "missing docs" outcome).
- `Task_FinalReject` – "Final Reject" (new branch from the decision gateway for the "ineligible" outcome).

**Removed tasks**
- `Task_RejectLoan` – "Reject Loan" (previously the only rejection branch; it has no counterpart with the same id/name in v2 and is not merely renamed — `Task_FinalReject` is a distinct, new task).

**Gateway-type changes**
- `Gateway_Decision` changed from an **exclusive** gateway (v1) to an **inclusive** gateway (v2). This is consistent with the move from two mutually exclusive outcomes (yes/no) to three potentially concurrent outcomes (eligible / missing docs / ineligible).
