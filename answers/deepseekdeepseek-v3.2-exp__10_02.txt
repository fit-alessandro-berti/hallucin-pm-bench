### **AtlasRemit: Cross-Border Payment Process RACI Table**

| Activity | Transaction Analyst | Regulatory Liaison | IT Support |
| :--- | :--- | :--- | :--- |
| **1. Receive and log customer transfer instruction** | **R** | - | - |
| **2. Screen transaction against sanctions list** | **R** | **I** | - |
| **3. Perform KYC Review on sender and beneficiary** | - | **R/A** | - |
| **4. Approve high-risk transactions** | - | **R/A** | - |
| **5. Release approved payment** | **R** | **I** | - |
| **6. Ensure SWIFT message is sent** | - | - | **R** |
| **7. Archive the case file** | **I** | **I** | **R** |

**RACI Key:**
- **R** = Responsible (Executes the task)
- **A** = Accountable (Owns the task and has veto/approval authority)
- **C** = Consulted (Provides input, two-way communication)
- **I** = Informed (Notified after task is done, one-way communication)
