Based on a comparison of the `Loan_Process` BPMN XML files, the following changes were identified:

### Added tasks
*   `Task_VerifyIdentity` (Verify Identity)
*   `Task_RequestDocs` (Request Additional Documents)
*   `Task_FinalReject` (Final Reject)

### Removed tasks
*   `Task_RejectLoan` (Reject Loan)

### Gateway-type changes
*   The gateway `Gateway_Decision` was changed from an **Exclusive Gateway** in v1 to an **Inclusive Gateway** in v2.
