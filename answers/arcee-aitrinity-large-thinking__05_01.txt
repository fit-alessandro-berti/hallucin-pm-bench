
Add a parallel gateway (id=Gateway_Parallel) after Start_Claim. Connect it to Task_Receive (replacing the existing direct flow) and to a new intermediate timer event (Timer_Escalation) with a 48-hour timer (PT48H). Add an execution listener to Task_Triage to set a process variable 'triageReached' to true when the task starts. When Timer_Escalation fires, route to an exclusive gateway (Gateway_Escalation). If 'triageReached' is false, escalate via a new end event (End_Escalation); if true, end the parallel path via another end event (End_OnTime). This monitors the SLA without adding tasks.
