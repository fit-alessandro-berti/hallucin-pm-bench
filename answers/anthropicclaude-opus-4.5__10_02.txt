# AtlasRemit Cross-Border Payment Process – RACI Matrix

Below is the RACI table based on the provided process description, using the mandated vocabulary throughout.

---

## RACI Table

| Task | Transaction Analyst | Regulatory Liaison | IT Support |
|------|:------------------:|:-----------------:|:----------:|
| **1. Receive transfer instruction and log in system** | R, A | I | — |
| **2. Screen payment against sanctions list** | R, A | C | — |
| **3. Perform KYC Review on sender and beneficiary** | I | R, A | — |
| **4. Approve high-risk flagged transactions** | I | R, A | — |
| **5. Release approved payment** | R, A | I | — |
| **6. Ensure SWIFT message is sent** | I | — | R, A |
| **7. Archive case file** | — | I | R, A |

---

## Legend

| Letter | Meaning |
|--------|---------|
| **R** | Responsible – Performs the task |
| **A** | Accountable – Ultimately answerable for completion |
| **C** | Consulted – Provides input before/during the task |
| **I** | Informed – Notified after task completion |

---

## Role Definitions (Mandated Vocabulary)

| Role | Description |
|------|-------------|
| **Transaction Analyst** | Handles receipt, logging, screening, and release of payment instructions |
| **Regulatory Liaison** | Conducts KYC Review activities and approves high-risk transactions |
| **IT Support** | Manages technical transmission (SWIFT) and case file archiving |
