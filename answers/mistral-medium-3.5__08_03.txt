### Goal:
Fire **T5** before **T3**.

4. To get a token in **pd**, we can either:
   - Fire **T2b** (which requires a token in **pa** and produces a token in **pd**), or

For **T5** to fire before **T3**, we need a token in **pd** *without* firing **T3**. The only way to do this is to fire **T2b**, which produces a token in **pd** directly from **pa**.

### Possible Firing Sequence:
1. Fire **T1**: Consumes p0, produces pa and pb.
   - Marking: pa = 1, pb = 1, others = 0.
2. Fire **T2b**: Consumes pa, produces pd.
   - Marking: pb = 1, pd = 1, others = 0.
3. Fire **T4**: Consumes pb, produces pe.
   - Marking: pd = 1, pe = 1, others = 0.
4. Fire **T5**: Consumes pd and pe, produces pf.
   - Marking: pf = 1, others = 0.

This sequence fires **T5** before **T3** (which has not fired yet).

### Shortest Firing Sequence:
The shortest sequence is: **T1, T2b, T4, T5**.

- After **T2b**: pd = 1, pb = 1.

Yes, it is possible for **T5** to fire before **T3**. One shortest firing sequence is:
