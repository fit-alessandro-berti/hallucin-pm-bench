# AtlasRemit Process – RACI Matrix

| Step | Task Description | Transaction Analyst | Regulatory Liaison | IT Support |
|:----:|------------------|:-------------------:|:------------------:|:----------:|
| 1 | Receive customer transfer instruction and log in system | **R** | I | - |
| 2 | Screen payment against sanctions list | **R** | I | - |
| 3 | Perform KYC Review on sender and beneficiary | C | **R/A** | - |
| 4 | Approve high-risk transaction (if flagged) | I | **R/A** | - |
| 5 | Release payment and ensure SWIFT message is sent | **R** | I | **R** |
| 6 | Archive case file and notify Regulatory Liaison | I | **I** | **R/A** |

**Legend**  
**R** = Responsible (performs the work)  
**A** = Accountable (ultimate ownership, approver)  
**C** = Consulted (provides input before decision)  
**I** = Informed (kept up-to-date after decision/action)
