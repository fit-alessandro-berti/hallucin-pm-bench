**Answer to Q1:** No, T5 cannot fire before T3.

**Explanation:** T5 requires a token in P2. Initially, P2 is empty. The only way to produce a token in P2 is via T1, which also creates a token in P1. T1 must fire first. T2 consumes P1 to create P3. T3 consumes P2 to create P4. Critically, P2 is never replenished. Therefore, T3 must consume the initial and only token in P2. Only after T3 fires can P4 eventually enable T4 and ultimately support T5. The logical order is constrained: T1 → (T2 and/or T3) → T4 → T5. T3 necessarily precedes the conditions for T5.
