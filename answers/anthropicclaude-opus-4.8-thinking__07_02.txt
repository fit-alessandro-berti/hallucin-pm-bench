## Summary of Changes

### • Added tasks
- **Task_VerifyIdentity** – "Verify Identity" (new task inserted between Start and Review Application)
- **Task_RequestDocs** – "Request Additional Documents" (new branch off the decision gateway)
- **Task_FinalReject** – "Final Reject" (new task with a distinct ID)

### • Removed tasks
- **Task_RejectLoan** – "Reject Loan" (present in v1, no longer exists in v2)

*(Note: Task_ApproveLoan persists with the same ID; its label change from "Approve Loan" → "Auto-Approve Loan" is a cosmetic rename and is excluded.)*

### • Gateway-type changes
- **Gateway_Decision**: changed from **Exclusive Gateway** (`exclusiveGateway`) → **Inclusive Gateway** (`inclusiveGateway`).
  - This allows multiple outgoing paths to be taken simultaneously; the branches also expanded from 2 (yes/no) to 3 (eligible / missing docs / ineligible).
